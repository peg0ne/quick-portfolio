> Halppp

A compiled language that is basically c++ with a different syntax. It also includes some of my most loved features in rust, c#, python and probably some more. It's an experimental language where I see what can be done and what feels nice to write. A key thing is that it tries to reduce the amount of {}, (), ; tokens in the language.
